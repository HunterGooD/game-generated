# Echoes of the Forgotten

**Кооперативный top-down action-roguelite на Godot 4.6.** Выбери класс, собери
синергичный билд и проверь, как глубоко он тебя пустит сквозь нарастающие волны,
подземелья и боссов — один или с друзьями.

> Этот README — канонический дизайн-док: суть игры, ключевая механика, петля
> геймплея, жизненный цикл и цель. Он честно разделяет **«как есть сейчас»** и
> **«цель»** (полный роадмап — в [`Feature.md`](Feature.md), раздел *Vision /
> Roadmap*). Технические доки: [`BUG.md`](BUG.md), [`Refactor.md`](Refactor.md),
> [`Server.md`](Server.md) (relay), [`EXPORT.md`](EXPORT.md).

## О чём игра

Ты — герой одного из 7 классов, спустившийся в забытые залы, где волны тварей и
мультифазовые боссы становятся всё опаснее. Сама по себе сила не спасёт — спасёт
**сборка**: то, как ты складываешь роль класса, трансформации скиллов и предметы в
единое синергичное целое. Игра вознаграждает не рефлексы как таковые, а удачно
собранный билд, проверенный на прочность эскалацией.

**Ради чего играть:** дойти глубже, чем позволил прошлый билд; раскрыть новую
синергию; пройти супербосса, которого в лоб не убить. И всё это — в кооперативе,
где классы дополняют друг друга.

## Ключевая механика: **Build × эскалация**

Сердце игры — **цикл «собери билд → проверь его глубиной»**. Билд складывается из
трёх слоёв, эскалация — это испытание, которое его ломает или подтверждает.

**Три слоя билда:**

1. **Роль (спек-путь)** — *цель:* на рубеже забега класс выбирает один из путей
   (caster / warrior / support), который переопределяет его роль. Сегодня роль
   фиксирована классом; гибкость уже доказана друидом (формы свопают набор скиллов).
2. **Трансформации скиллов** — уникалки и level-up меняют САМО поведение скилла
   (Fire Wall → Ледяная стена, Raise Knight → Проклятое поле, Bear Form → Лютоволк).
   Это не «+10% урона», а смена роли скилла.
3. **Предметы** — base + аффиксы + сеты + уникалки (*цель:* сокеты и камни усиления
   с риск-наградой). Сеты и уникалки «зовут» конкретные сборки.

**Эскалация** — волны и (в перспективе) комнаты карты, которые всё жёстче: больше
врагов, выше статы, элиты с аффиксами, мультифазовые боссы. Кульминация карты —
**супербосс**, который служит экзаменом: убить его без синергии нельзя.

> Одной фразой: **игра — это головоломка сборки, которую ты решаешь действием.**

## Core gameplay loop

Две вложенные петли.

**Минута-к-минуте (бой):**
```
двигайся / уворачивайся  →  базовая атака + 4–5 скиллов (КД, мана)
        →  читай телеграфы врагов и боссов  →  собирай лут и золото
        →  набил уровень → выбор апгрейда  →  следующая пачка / босс
```

**Забег-к-забегу (мета-петля):**
```
выбор класса (+ спек-путь*)  →  волны / комнаты  →  мерчант  →  боссы
        →  СУПЕРБОСС*  →  выбор: идти глубже (↑сложность) | завершить → лобби*
        →  потратить очки в дереве пассивок*  →  новый, более сильный забег
```
`*` — целевые элементы из роадмапа; см. жизненный цикл ниже.

## Жизненный цикл

```
              ┌─────────────────────────────────────────────┐
              │  ЛОББИ / выбор класса (+ спек-путь*)          │
              └───────────────┬─────────────────────────────┘
                              ▼
        ┌─────────────────────────────────────────────────────┐
        │  ЗАБЕГ — комнаты/волны:                              │
        │   волна ──(каждая 3-я)──> мерчант                    │
        │     │                                                │
        │     └──(каждая 5-я)──> босс                          │
        │   *данж / ивент / boss-rush комнаты (цель)           │
        └───────────────┬───────────────────────┬─────────────┘
                        ▼                         ▼
                 СУПЕРБОСС* (конец карты)     СМЕРТЬ
                        │                         │
            ┌───────────┴───────────┐            ▼
            ▼                       ▼        game_over
   Продолжить (↑сложность*)   Завершить → ЛОББИ*  → (рестарт / меню)
            │                       │
            └──> новая карта        └──> дерево пассивок* / анлоки*
```

**Как есть сейчас:** забег = бесконечные волны до смерти. Мерчант каждую 3-ю волну
(через портал), босс каждую 5-ю (4 уникальных мультифазовых босса). Смерть: соло —
мгновенный конец; кооп — *downed* + кровотечение 15 c + ревайв тиммейтом, все мертвы
→ game_over (рестарт или меню). Между забегами не сохраняется ничего.

**Цель (`Feature.md`):** забег = сгенерированная карта из комнат (волны / данж в духе
Diablo / ивенты / boss-rush / мерчант) с супербоссом в конце; после победы — выбор
«продолжить на более сложную карту» или «вернуться в лобби-хаб» и вложить очки в
дерево пассивок. Мета-прогрессия превращает смерть из тупика в шаг.

---

## Цель и условия победы/поражения

| | Сейчас | Цель |
|---|---|---|
| **Победа** | нет win-условия (endless) | убить супербосса карты → этап пройден |
| **Метрика** | highest wave (survival-score) | глубина карт + ascension-тир |
| **Поражение** | смерть (соло мгновенно, кооп после bleed-out) | то же + потеря прогресса забега, но мета сохраняется |
| **«Дальше»** | рестарт с нуля | продолжить глубже ИЛИ закрепить мету в лобби |

## Классы и роли

7 классов, у каждого 4–5 скиллов и своя фантазия. Спек-пути (*цель*) дадут каждому
ролевую развилку caster / warrior / support; druid уже сейчас — доказательство
концепции (может заменить трансформации и играть как кастер, так и как зверь).

| Класс | Фантазия | Скиллы (слоты 1–4/5) |
|---|---|---|
| **Mage** | Стихийный артиллерист | Fire Wall · Ice Bolt · Chain Lightning · Meteor |
| **Barbarian** | Ближний бёрст и контроль | Whirlwind · Leap Slam · Battle Cry · Earthquake |
| **Rogue** | Ловушки и веер клинков | Caltrops · Smoke Bomb · Poison Vial · Fan of Knives |
| **Druid** | Оборотень-перевёртыш | Wolf/Bear/Eagle Form · Stone Armor · Summon Spirit |
| **Necromancer** | Повелитель миньонов | Raise Skeleton · Raise Knight · Blood Pact · Death Pulse |
| **Hexen** | Проклятия и детонации | Hex Mark · Blood Whip · Soul Tether · Crimson Ritual |
| **Stormcaller** | Молнии и статический заряд | Chain Bolt · Storm Step · Sky Strike · Static Discharge |

Боссы (каждая 5-я волна): The Crimson Matron, The Hellgate Sovereign, The
Shadewitch, The Lich Empress.

## Статусы, баффы и щиты

Боевые эффекты — общая, переиспользуемая система: скиллы возвышений (спек-путей)
накладывают их через единые методы, а не хардкодят логику. Все дебаффы на врагах
**host-authoritative** (тикают на хосте/в соло; на клиенте-puppet'е не считаются).

**Дебаффы (на враге)** — `enemy.gd`:

| Статус | Метод | Эффект |
|---|---|---|
| Slow | `apply_slow(t, mult)` | замедление move speed |
| Burn 🔥 | `apply_burn(t, dps)` | огненный DoT, метит стихию «fire» |
| Chill / Freeze ❄ | `apply_chill(t, stacks)` | стак-замедление; на 4 стаках — заморозка |
| Bleed 🩸 | `apply_bleed(t, dps)` | физический DoT |
| Poison ☠ | `apply_poison(stacks, t, dps)` | стак-DoT до 10 → Necrotic (+уязвимость) |
| Vulnerable / Armor Break | `apply_vulnerable(t, amp)` | +% входящего урона |
| Taunt | `apply_taunt(node, t)` | форс-цель (враг бьёт указанный объект) |
| Elemental Fracture | `mark_element("fire/frost/storm")` | 3 стихии за 5с → «расколот»: +урон + взрыв при смерти |
| Curse | `add_curse_stack()` | счётчик проклятий (Hexen-комбо) |

**Баффы / защита (на игроке)** — `player.gd`: `apply_buff` (урон+скорость),
`apply_stealth` (невидимость), `apply_aura` (пати-аура: урон+DR), `apply_evasion`
(шанс уклонения), Flameblade / Frenzy / Backstab-окна, `grant_funeral` (анти-смерть).

**Щиты — работают ✅** (две независимые механики):
- **Shield-пул** `shield_hp` — `add_shield(amount, cap)`; поглощает урон **до HP** в
  `receive_damage_payload` (Battlemage burn-shield, Chronomancer Stasis Star, пати-
  щиты Warchief/Conductor/Gravebinder и т.д.).
- **Stone Armor charges** `stone_armor_charges` — гасит **по одному удару целиком** за
  заряд (друидовский Stone Armor, Frost Guard, Hide of the Beast).

### Отображение статусов (HUD + над врагами)

Активные статусы рисуются рядами **квадратиков** с эффектом **циферблата** —
радиальная заливка «отматывает» оставшееся время (полный → пустой по часовой).

- Шейдер: [`assets/shaders/status_dial.gdshader`](assets/shaders/status_dial.gdshader)
  (`progress` = остаток времени; `tint`/`use_icon` — заглушка-квадрат или будущая
  картинка). Виджет-ряд: [`scripts/ui/status_icons.gd`](scripts/ui/status_icons.gd).
- **Игрок:** ряд под маной (буффы/щиты с буквенными метками). **Враги:** маленький
  ряд над врагом (его дебаффы). Источник данных — `get_active_statuses()` на игроке
  и враге.
- *Сейчас* квадрат — заглушка (цвет статуса + буква). *Позже*: вместо квадрата —
  картинка статуса, а шейдер-циферблат полупрозрачным слоем сверху — менять можно
  без правок раскладки (`use_icon` + Texture).

## Прогрессия

**В забеге (есть):** level-up → выбор из стат-награды / skill-модификатора /
transform-уникалки (фильтруется по классу); лут с аффиксами, сетами (Ironclad,
Voidweave, Shadowsilk) и уникалками; золото у мерчанта.

**Цель:** сокеты и камни усиления с риск-наградой (`Feature.md` V5); персистентное
**дерево пассивок** на каждый класс, очки тратятся в лобби (V2); мета-валюта и
анлоки классов/уникалок/перков (V7).

## Реиграбельность: пиллары (роадмап)

Полностью — в [`Feature.md`](Feature.md) → *Vision / Roadmap*. Кратко:

- **V1 Спек-пути** — ролевая гибкость каждому классу (ближайшее).
- **V2 Дерево пассивок** — персистентная мета-прокачка per-class.
- **V3 Генерация карты** (Slay-the-Spire) — комнаты-ноды, ≤2 столба наград, супербосс.
- **V4 Ивенты** — оживить готовый стаб beacon'ов (алтари, сокровища, рулетка, ритуал).
- **V5 Сокеты + камни + gamble** — Diablo-like апгрейд экипировки с риском.
- **V6 Ран-модификаторы / элиты** — качественная сложность вместо «больше HP».
- **V7 Мета + лобби-хаб** — сейв, валюта, анлоки (фундамент удержания).
- **V8 Контент** — больше врагов/боссов/биомов; in-form уникалки друида.

## Новые идеи (свежий брейншторм)

Не дублируют роадмап — кандидаты на перенос в [`Feature.md`](Feature.md):

- **Сигнатурные дропы боссов** — каждый босс роняет тематическую build-defining
  уникалку → причина фармить конкретного босса под сборку.
- **Rest site (костёр a-la StS)** между комнатами: «хил / апгрейд предмета / реролл
  скилла» — выбор, а не только бой.
- **Ковенанты сложности** (Hades): добровольные ограничения («без хила», «двойной
  спавн») за повышенный тир награды.
- **Runewords / инскрипции** поверх сокетов — именованный эффект за паттерн камней
  (глубокий крафт Diablo 2).
- **Кодекс синергий** — журнал открытых комбо; мета-цель «собрать все».
- **Командные ауры минионов** (necro/druid): стойка aggressive/defensive/follow.
- **Интерактив окружения** в данж-комнатах: взрывные бочки и ловушки, в которые
  заманиваешь врагов.
- **Карты-модификаторы** (D3 rifts / PoE maps): предмет-карта задаёт модификаторы и
  босса следующей комнаты — игрок сам крутит риск-награду маршрута.
- **Еженедельный мутатор** — общий сид + фикс-набор модификаторов, повод возвращаться.

## Кооператив

Модель **host-authoritative**: всё геймплейно-важное (враги, миньоны, урон, волны,
лут) симулирует ХОСТ и реплицирует остальным «пуппетами»; клиент шлёт запрос/ввод,
хост решает. Реализованы: downed + ревайв тиммейтом, host-authoritative
summon-sync (миньоны реально танчат и бьют), кооп-пауза по числу подключённых.
Транспорт — Rust relay-сервер со строгим enum-протоколом (см. [`Server.md`](Server.md)).
Спек-пути (*цель*) дадут командные роли: танк-аура, хил-тотем, масс-контроль.

## DevConsole
Дев-консоль готова — теперь элиты (и всё остальное) тестируются прямо из игры.
Тоггл по ~. Команды:

| Команда                                                       | Что делает                                                                   |
|---------------------------------------------------------------|------------------------------------------------------------------------------|
| spawn_elite <type> <affixes>                                  | спавн элиты; аффиксы через запятую (swift,brutal) или пусто = случайный ролл |
| spawn_enemy <type> <count>                                    | N обычных врагов                                                             |
| spawn_boss <id>                                               | босс (тест HSM-фаз)                                                          |
| give_levels <n> · give_gold <amt> · give_item <count> <wave>  | выдать уровни / золото / предметы (роллит через LootRoller)                  |
| list_affixes · list_enemies                                   | список id (чтобы знать, что спавнить)                                        |
| heal · kill_all                                               | полный хил / убить всех                                                      |
